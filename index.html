统计日期

Share

内容简介 : 较难

  • 出题人 : 朱俊老师
  • 试题内容 — 统计日期

题目描述

从1900年开始计算,统计未来n年之内每个月13号是星期一,星期二…星期日的频率。

注解 : 从1900年1月1日开始算起一直到1900+n-1年的12月31日号

输入

输入年数 n

(1 <= n <= 300)

输出

依次输出从星期一到星期日是在13号出现的次数

样例输入

40

样例输出

69 68 70 68 69 68 68

提示

  • 我的代码
#include <iostream>
#include <cstring>

using namespace std;

int cnt[8];

int main() {
    int n;
    cin >> n;
    int k = 1;
    memset(cnt, 0, sizeof cnt);
    for(int i = 0; i < n; i++)
        for(int j = 1; j <= 12; j++) {
            int day = (12 + k) % 7;
            cnt[day ? day : 7]++;
            if(j == 4 || j == 6 || j == 9 || j == 11) k += 30;
            else if(j == 2) {
                int year = 1900 + i;
                if((year % 400) == 0) k += 29;
                else if((year % 4) == 0 && year % 100) k += 29;
                else k += 28;
            } else k += 31;
        }
    for(int i = 1; i < 8; i++) cout << cnt[i % 7 ? i % 7 : 7] << ' ';
    puts("");
    return 0;
}

机器人

Share

内容简介 : 较难

  • 出题人 : 朱俊老师
  • 试题内容 — 机器人

题目描述

现假设笛卡尔坐标系上有一个机器人，其坐标为(0, 0)，现在，这个机器人想要到达(x, y)这个位置。
这个机器人每秒都会执行一个指令，指令的列表如下：
指令I：向上移动，即坐标从(i, j)变为(i, j + 1)
指令II：向右移动，即坐标从(i, j)变为(i + 1, j)
指令III：向下移动，即坐标从(i ,j)变为(i, j - 1)
指令IV：向左移动，即坐标从(i, j)变为(i - 1, j)
指令V：原地停留一秒，即不改变其坐标。
此外，对于机器人来说，相同的指令不能连续执行两次。
现在，我们想要知道这个机器人从(0, 0)到(x, y)，最少要多少秒？

输入

输入共1行，有2个非负整数x和y，表示要到达的目的地坐标是(x, y)

输出

输出从(0, 0)到(x, y)最少需要的秒数。

样例输入

0 0

样例输出

0

提示

样例解释#1：

因为机器人就在(0, 0)坐标，所以无需任何操作就可到达目标坐标，故只需0秒。

样例输入#2

2 0

样例输出#2

3

样例解释#2：

指令序列如下：II,V,II。先往右走1个单位，因为不能有两个连续的向右命令，故需要原地停留一秒，再向右走一个单位，故总共经过3秒。

数据约定：

对于100%的数据：

0 <= x, y <= 104

内容简介 : 较难

题目描述

小紫拥有一块带n个内存插槽的超级主板，每个内存插槽都可以插一根内存条，每根内存条的内存都只能是1~n中的一个整数。小紫想要一块理想的超级主板，那么，什么是理想的超级主板呢？
首先，这块主板上所有的内存插槽都必须被使用，即每个内存插槽都必须有一根内存条。
其次，如果我们给每个插槽按顺序编上号，那么，这些内存插槽里的内存条之间必须满足其中之一的条件：

1) 从第二根内存条开始，每根内存条的内存不得超过上一根内存条的内存。
2) 从第二根内存条开始，每根内存条的内存不得小于上一根内存条的内存。
现在，假设我们已知插槽的数量，那么，有多少种组合方案能够让一块普通的超级主板变成理想超级主板？

输入

输入数据有多组。
每行有一个正整数n，表示超级主板的内存插槽数量是n。

输出

对于每组输入，输出一个正整数，表示符合理想超级主板条件的内存条组合方案数。
因为方案数可能过大，所以请你输出方案数与109+7取模后的结果。

样例输入

2

样例输出

4

提示

样例解释：

如果内存插槽数量为2，则可能的组合方案如下：

(1) 两个插槽都插内存为1的内存条，满足题目中的条件1和条件2.

(2) 两个插槽都插内存为2的内存条，满足题目中的条件1和条件2.

(3) 第一个插槽插内存为1的内存条，第二个插槽插内存为2的内存条，满足题目中的条件2.

(4) 第一个插槽插内存为2的内存条，第二个插槽插内存为1的内存条，满足题目中的条件1.

因为一共有4种组合方式，故输出4.

对于100%的数据：

1 <= n <= 105

题目描述

你是一名英雄，你现在要去寻找你的宝剑，现在有一个R*C的迷宫，你在字符S的位置，宝剑在字符E的位置，字符#表示陷阱，字符.表示可以通过，你在 1 个单位时间内可以从当前的位置走到它上下左右四个方向上的任意一个位置，但不能走出地图边界。问你最短时间如何找到自己的宝剑？

#include <iostream>

大鱼吃小鱼，小鱼吃虾米

Share

内容简介 : 较难

  • 出题人 : 朱俊老师
  • 试题内容 — 大鱼吃小鱼，小鱼吃虾米

题目描述

大鱼吃小鱼，小鱼吃虾米

在自然界之中存在这样一种大鱼吃小鱼，小鱼吃虾米的弱肉强食的关系。

我们可以形式化进行定义 M 吃 X , X 吃 Y，Y 吃 M。这样就形成了一种环状关系。

现有如下问题 :

M,X,Y每一个代表一种生物，但是并不能确切的确定他们的对应关系。

<1>. 如若 “A M X”,那么M 和 X就是属于同一阶层的生物。

对于n个生物我们都可以使用上述两种定义进行说明。

但是根据常识若满足以下三种情况，那么这种定义就是错误的。

<2>. 若当前的定义中当前的M 和 X 比生物总个数n大，那么就是错误的;

你需要根据上述规则，给定你n个生物，k个定义,

输出错误的定义的个数。

数据范围:

1<= n <= 10000;

输入

给定n和k,都是整数,中间用空格隔开

接下来每一行一个op,M,X,中间用空格隔开，op代表定义的两种类型:

若 op == ‘A’ 那么表示 M，X 是同一阶层的生物。

若 op == ‘B’ 那么表示 M 和 X就是属于M是X的上一层生物，M可以吃掉X

输出

错误定义的个数

样例输入

50 5
A 55 1
B 1 2
B 2 3
A 2 1
B 2 2

样例输出

3

提示

  • 我的代码
#include <iostream>
#include <cstdio>
#include <cstdlib>
#include <cstring>
#include <string>
#include <cmath>
#include <algorithm>
#include <queue>
#include <vector>
#include <stack>
#include <map>
#include <set>
using namespace std;

int n;
int k;

char op;
int x;
int y;

const int N = 5e4 + 10;
const int M = 3;

int parent[N];
int d[N];

void init() {
    for (int i = 0; i <= n; i++) {
        parent[i] = i;
        d[i] = 0;
    }
}

int find(int x) {
    if (x != parent[x]) {
        int oldParent = parent[x];
        parent[x] = find(parent[x]);
        d[x] = (d[x] + d[oldParent]) % M;
    }
    return parent[x];
}

bool D1(int x, int y) {
    int p1 = find(x);
    int p2 = find(y);

    if (p1 == p2) {
        return d[x] % M == d[y] % M;
    }

    parent[p2] = p1;
    d[p2] = ((d[x] - d[y]) + M) % M;
    return true;
}

bool D2(int x, int y) {
    int p1 = find(x);
    int p2 = find(y);

    if (p1 == p2) {
        return d[x] % M == (d[y]+1) % M;
    }

    parent[p2] = p1;
    d[p2] = ((d[x]-d[y]-1) + M) % M;
    return true;
}

int main(void) {
    int res = 0;

    cin >> n;
    init();

    cin >> k;
    while (k--) {
        cin >> op >> x >> y;
        if (x > n || y > n) {
            res += 1;
        } else {
            if (op == 'A') {
                if (D1(x, y) == false) {
                    res += 1;
                }
            }
            if (op == 'B') {
                if (D2(x, y) == false) {
                    res += 1;
                }
            }
        }
    }

    cout << res << endl;

    return 0;
}

超级吸铁球

Share

内容简介 : 较难

  • 出题人 : 朱俊老师
  • 试题内容 — 超级吸铁球

题目描述

小紫在玩一款益智游戏《超级吸铁球》。
在这款游戏中，每一关都有一定数量的球，你唯一能进行的操作就是选中平面上的任意一个球，然后以该球为中心，所有和该球的曼哈顿距离小于等于一个定值的球都会被吸引过来，因为无视碰撞体积，故范围内的所有球都会被吸引到中心点上。
一关的达成条件就是在限定的操作次数内，将所有的球都吸引到同一坐标。此外，这款游戏还有一个奖励机制，每一关都有一个理论上最少操作次数，只要你能够以最少操作次数完成本关，游戏就会奖励你限定的游戏皮肤。
这对于小紫来说诱惑太大，但是她并不是每一关都能达到最少操作次数，因此她想求助聪明的你来帮助她达成这个目标。
现在我们假设存在一个平面，平面上有n个位置均不同的球，坐标依次为(x1, y1), (x2, y2), …, (xn, yn)。
同时我们指定一个最大吸引距离k。
你需要做的就是求出将所有点吸引到同一坐标的最少操作次数。游戏是必须有解的，但是本题不一定有解，对于无解情况，请你输出-1.

如左图所示，我们选中了图中央的球，以该球为中心的红色菱形区域内的所有球，都会被吸引到同一个点，就如右图所示。

输入

第一行包含两个整数n和k，n表示球的数量，k表示最大吸引距离。
第2至n+1行，每行2个整数xi和yi，表示球i的坐标是(xi, yi)

输出

输出将所有球移动到同一坐标的最少操作次数，如果不可能，则输出-1.

样例输入

3 2
0 0
1 1
2 2

样例输出

1

提示

数据约定：

对于100%的数据：
4
40

import java.math.*;
import java.util.*;
public class Main {
    public static void main(String[] args){

        Scanner sc = new Scanner(System.in);

        int cnt = sc.nextInt();

        BigInteger mto = sc.nextBigInteger();

        PII p[] = new PII[cnt];

        for (int k = 0; k < cnt; k++) {

            p[k] = new PII(sc.nextBigInteger(), sc.nextBigInteger());

        }
        int myanswer = -1;

        for (int m = 0; m < cnt; m++) {

            boolean mresflag = false;

            for (int n = 0; n < cnt; n++) {

                BigInteger res = p[m].x.subtract( p[n].x ).abs();

                res = res.add( p[m].y.subtract(p[n].y).abs() );

                if (res.compareTo(mto) == 1) {

                    mresflag = true;

                    break;
                }
            }
            if (mresflag == false){

                myanswer = 1;

                break;
            }
        }
        System.out.println(myanswer);
    }
    static class PII {

        public BigInteger x, y;

        public PII(BigInteger first, BigInteger second) {
            this.x = first;
            this.y = second;
        }
    }
}

题目描述

大白是小店管理员,他在想如何提升自己店的销售利润,一天小Q把店铺里的商品以及过期时间告诉了大白。你能帮助大白合理安排每天的商品销售吗?每天只能售卖一件物品哦!

说明: 每一件物品都有wi以及过期时间di,每天只能卖一件物品,并且过期的物品不能再卖掉了。如何安排每天卖掉的物品使得小店的利润收益最大?

输入

输入包含多组测试用例

每组测试用例，以输入整数n开始，接下来输入n对wi和di，分别代表第i件物品卖出的利润和过期时间。

在输入中，数据间可以隔着多个空格，输入一直到文件的最终结尾时终止输入，要求保证数据的正确性。

输出

输出格式:

要求包含每组物品的一个最大收益值。 每个结果占一行。

数据范围说明:

样例输入

4  50 2  10 1   20 2   30 1

7  20 1   2 1   10 3  100 2   8 2
   5 20  50 10

样例输出

80
185

#include<iostream>
#include<algorithm>
#include<cstdio>
#include<vector>
#include<cstring>
using namespace std;
typedef pair<int,int> PII;
const int N = 1e4+233;
vector<PII> res;
int f[N];
int dfs(int x)
{
    if(f[x] == x) return x;
    return f[x] = dfs(f[x]);
}
int main()
{
    int n;
    while(cin >> n)
    {
        int ans = 0;
        res.clear();
        int mymax = 0;

        for(int i = 1; i <= n; i++)
        {
            int u,v;
            cin >> u >> v;
            res.push_back({-u, v});
            mymax = max(mymax, v);
        }

        for(int i = 0; i <= mymax; i++)
        {
            f[i] = i;
        } 
        sort(res.begin(), res.end());
        for(int i = 0; i < res.size(); i++)
        {
            int v = -res[i].first, e = res[i].second;
            int pos = dfs(e);
            if(pos > 0)
            {
                ans += v;
                f[pos] = pos - 1;
            }
        }
        cout << ans << endl;
    }
}
