堆石头

Share

内容简介 : 较难

  • 出题人 : 朱俊老师
  • 试题内容 — 堆石头

题目描述

有n堆不同种类的石头，你需要把所有堆石头合成一堆。每一次合并，你可以把两堆石头合并到一起，消耗的体力等于两堆石头的重量之和。可以看出，所有的石头经过n-1次合并之后，就只剩下一堆了。你在合并石头时总共消耗的体力等于每次合并所耗体力之和。求最小的体力消耗

输入

输入包括两行，第一行是一个整数n，表示石头的种类数。

第二行包含n个整数，用空格分隔，第i个整数ai是第i种石头的数目。

输出

输出包括一行，这一行只包含一个整数，也就是最小的体力耗费值。

样例输入

3 
1 2 9

样例输出

15

提示

数据约定：

对于100%的数据：

1≤n≤10000,1≤ai≤20000

  • 我的代码
#include<iostream>
#include<algorithm>
#include<queue>
#include<cstdio>
#include<vector>
using namespace std;

int main(){
    int n;
    priority_queue<int,vector<int>,greater<int>> heap;
    cin>>n;
    for(int i=0;i<n;i++){
        int m;
        cin>>m;
        heap.push(m);
    }

    int res=0;
    while(heap.size()>1){
        int a=heap.top();heap.pop();
        int b=heap.top();heap.pop();
        res+=(a+b);
        heap.push(a+b);
    }

    cout<<res;
    return 0;
}

采药人

Share

内容简介 : 较难

  • 出题人 : 朱俊老师
  • 试题内容 — 采药人

题目描述

小紫想成为世界上最伟大的医师。今天，她碰到了一个难题。一位有名的医师把她带到一个到处都是草药的山洞里对她说：“这个山洞里有一些不同的草药，采每一株都需要一些时间，每一株也有它自身的价值。如果我给你一段时间，在这段时间里，你可以采到一些草药。我相信你是一个聪明的孩子，应该可以让采到的草药的总价值最大。”

如果你是小紫，你能完成这个任务吗？

输入

第一行有两个整数T，M。T表示总共能够用来采药的时间，M表示山洞里的草药数量。
第2~M+1行，每行有两个整数Ti和Wi，Ti表示采摘第i株草药需要的时间，Wi表示第i株草药的价值。

输出

输出在规定的时间内可以采到的草药的最大总价值。

样例输入

70 3
71 100
69 1
1 2

样例输出

3

提示

数据约定：

对于100%的数据：

1 <= T, Ti <= 1000

1 <= M, Wi <= 100

import java.util.*;
import java.math.*;
import java.io.*;

public class Main {

    static BufferedWriter out;
    static long ans = 0x3f3f3f3f;
    static long modulo = (long)1e9 + 7;


    public static void main(String[] args) throws IOException {
        //StreamTokenizer in = new StreamTokenizer(new BufferedReader(new InputStreamReader(new FileInputStream("2020.txt"))));
        out = new BufferedWriter(new OutputStreamWriter(System.out));
        Scanner in = new Scanner(System.in);
        int t = in.nextInt();
        int m = in.nextInt();
        int weight[] = new int[m], value[] = new int[m];
        int dp[] = new int[t + 1];
        for (int i = 0; i < m; i++) {
            weight[i] = in.nextInt();
            value[i] = in.nextInt();
        }
        for (int i = 0; i < m; i++) {
            for (int j = t; j >= 0; j--) {
                if (j >= weight[i]) {
                    dp[j] = Math.max(dp[j], dp[j - weight[i]] + value[i]);
                }
            }
        }
        out.write(String.valueOf(dp[t]));
        out.flush();
    }
}

内容简介 : 基础题

  • 出题人 : 林航威
  • 试题内容 — 帮帮光头强

题目描述

光头强的老板因光头强办事不力想开了他，但在光头强的要求下又给了他一个机会让他证明自己的伐木能力。老板只允许光头强砍倒单行树木。光头强的考验过程如下：光头强设置一个高度参数h米，伐木机升起一个巨大的锯片到高度h，并锯掉所有的树比h高的部分（当然，树木不高于h米的部分保持不变）。光头强可以树木被锯下的部分。例如，如果一行树的高度分别为20，15，10和16，光头强把锯片升到15米的高度，切割后树木剩下的高度将是15，15，10和15，而光头强将从第1棵树得到5米，从第4棵树得到1米，共得到6米木材。老板要考验光头强的伐木效率和计算能力，你能帮助光头强找到伐木机锯片的最大的整数高度H，使得他能得到木材至少为M米。如果再升高1米，则他将得不到M米木材。

输入

输入一共有2行。

第1行是两个正整数n和m，n表示树木的数量，m表示老板要求的木材总长度。

第2行有n个整数，第i个整数表示第i棵树的高度为hi。

输出

输出砍树的最高高度。

样例输入

10 30
15 27 9 32 29 7 7 21 8 34

样例输出

23

提示

数据约定：

对于100%的数据：

1 ≤ n ≤ 106

1 ≤ m ≤ 2 * 109

Σhi ≤ 109

题目保证一定有解。

游玩栖霞山

Share

内容简介 : 基础题

  • 出题人 : 林航威
  • 试题内容 — 游玩栖霞山

题目描述

学校组织计算机专业的学生去栖霞山游玩，要用校车带同学们去栖霞山。但是校车有限只有一辆所以要租借校外的大巴车带同学们去。假设租借的车和校车的载客量一样，都为50；请问最少需要租借几辆大巴车才能带所有同学去栖霞山（以班级为单位即一个班的学生必须在同一辆车内，一辆车可以载多个班级, 班级的人数不大于50）。

输入

输入总共有2行。

第1行是一个正整数n，表示班级的数量。

第2行有n个正整数，分别表示第i个班级的人数为Ci，数字之间使用空格隔开。

输出

输出最少需要租借的车辆数。

样例输入

5
20 30 40 50 10

样例输出

2

提示

数据约定：

对于100%的数据：

1 ≤ n ≤ 100

1 ≤ Ci ≤ 50

  • 我的代码
#include<stdio.h>
int main()
{
    int n;int a[110]={0};
    scanf("%d",&n);
    for(int i=0;i<n;i++)
        scanf("%d",&a[i]);
    for(int i=0;i<=n-1;i++)
        for(int j=0;j<n-i-1;j++)
            if(a[j]>a[j+1])
            {
                int t=a[j];
                a[j]=a[j+1];
                a[j+1]=t;
            }
    int num=0,sum=50,l=0,r=n-1;
    while(l<=r)
    {
        if(a[l]+a[r]>sum)
        {
            sum=sum-a[r];
            r--;
            num++;
            sum=50;
        }
        else
        {
            sum=sum-a[l]-a[r];
            l++;r--;
        }
    }
    printf("%d",num);
}

内容简介 : 基础题

  • 出题人 : 陶宇杰
  • 试题内容 — 试卷整理

题目描述

期中考试结束了。请你帮老师按照分数高低(升序或者降序)整理n份试卷，以便统计同学们考试的排名。分数相同的试卷算作同一个排名。

输入

输入一共有三行。

第一行为试卷的数量n。

第二行是排序的顺序seq，seq=1表示升序输出，seq=-1表示降序输出。

第三行共有n个非负整数，其中第i个非负整数表示第i份试卷的分数为xi，每个非负整数之间使用”,”隔开。

输出

输出试卷整理后的结果，分数之间使用空格隔开。

样例输入

6
1
90,89,87,88,90,88

样例输出

87 88 89 90

提示

数据约定：

1 ≤ n ≤ 1000

0 ≤ xi ≤ 100

seq = -1, 1

注意：

  1. 存在分数相同的试卷

  2. 注意输出格式

内容简介 : 基础题

  • 出题人 : 杨宇
  • 试题内容 — 进制转换 2.0

题目描述

小紫的老师给小紫出了一道进制转换的题目：将十进制数转换成二进制数表示，这对于学过专业导论的小紫来说非常简单，小紫按照导论书上给出的算法思路，很容易就写出了对应的C语言代码，并成功地完成了题目。现在，对于参加天梯赛的你们来说，这显然也是小菜一碟，所以我们要加大力度。

第2行是一个正整数m，m表示用m进制输出数s。

将数s转换成m进制然后输出。

样例输入

10 1000
16

样例输出

3E8

提示

数据约定：

对于50%的数据：

2 ≤ n, m ≤ 16, n ≠ m

-105 ≤ s ≤ 105

对于100%的数据：

2 ≤ n, m ≤ 36, n ≠ m

-1018 ≤ s ≤ 1018

long TenNum(char a[], long long B)
{
    int len, i, num;
    long sum = 0;     
    len = strlen(a);
    for (i = 0; i < len; i++)
    {
        if (a[i] >= '0' && a[i] <= '9')

        else if (a[i] >= 'A' && a[i] <= 'Z')

}
